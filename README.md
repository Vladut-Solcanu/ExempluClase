# Joc Figura Geometrică

Acest program este o aplicație tip joc care permite utilizatorului să introducă un număr de laturi și să afle ce tip de figură geometrică corespunde. În plus, programul generează automat mai multe figuri cu un număr aleator de laturi și afișează tipul acestora.

Funcționalități:
- Introducere manuală a numărului de laturi și afișarea tipului figurii.
- Generare automată de figuri cu număr aleator de laturi.
- Afișarea tipului corespunzător fiecărui număr de laturi generat.

Exemplu de rulare:
Introduceți numărul de laturi: 3
Figura cu 3 laturi este: Triunghi

Generare automată de figuri:
Figura 1: 5 laturi -> Pentagon
Figura 2: 0 laturi -> Punct
Figura 3: 3 laturi -> Triunghi
Figura 4: 7 laturi -> Poligon cu 7 laturi
Figura 5: 4 laturi -> Patrulater
